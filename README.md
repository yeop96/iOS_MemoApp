## #MEMO:RE

### 간단한 iOS 메모장 앱
 
#### 기능

1. 작성한 메모 리스트 작성순 정렬
2. 메모 작성 및 수정 (개행하기전까지 제목 미리보기 가능)
3. 메모 고정 및 삭제
4. 메모 제목 및 내용으로 검색
5. 작성 메모 공유
